# Todo app

This is a unioned repo to run both front, back and mongo in docker

## Steps to install

1. Clone this repo

```bash
git clone --recurse-submodules https://github.com/Yarik7610/todo.git
```

2. See instructions for back part in README.md
3. Run next command:

```bash
docker compose up
```

4. Go to localhost:3000
